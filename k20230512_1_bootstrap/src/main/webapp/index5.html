    최진욱의 코딩은 다른 사이트와 차원이 다른 깔끔한 구성을 보여줍니다.

    자세히

    폭 넓은 강사진

    최진욱의 코딩은 뛰어난 능력을 가진 강사진으로 뽑습니다.

    자세히

    핵심적인 강의

    최진욱의 코딩의 강의는 실제 개발에 있어서 반드시 필요한 핵심적인 내용을 중심으로 다룹니다.

    자세히

      최신 강의 목록

    C언어 기초 프로그래밍 강의NEW

    C언어 강의는 기초 프로그래밍 강의입니다.
    처음 프로그램을 접하는 입문자가 듣기 적합한 강의로
    강의료는 무료이며 C언어 기초 프로그래밍 강의는 총 20강으로 구성됩니다.

    Java 기초 프로그래밍 강의NEW

    Java언어 강의는 객체지향 프로그래밍의 기본을 알려주는 강의입니다.
    처음 프로그램을 접하는 입문자가 듣기 적합한 강의로
    강의료는 무료이며 Java 기초 프로그래밍 강의는 총 20강으로 구성됩니다.

    Android 프로그래밍 강의NEW

    Android 강의는 Java 또는 Kotlin을 활용하여 Android 어플리케이션을 개발하는 프로그래밍 강의입니다.
    속도감있는 강의로 앱 개발 입문자가 빠르게 입을 만들어 보면서 기본 개념을 탄탄히 다지도록 합니다.
    강의료는 100,000이며 Android 기초 프로그래밍 강의는 총 40강으로 구성됩니다.
